--Ambiente--
Instalar o Express
npm install express pg sequelize dotenv cors
npm install express sequelize pg

Instalar o Postgre
npm install sequelize pg-hstore

Installa o Swagger
npm install swagger-ui-express jsdoc
npm install swagger-jsdoc

Instalar jest
npm install --save-dev jest supertest


Monitor de alterações
npm install --save-dev nodemon

Cria ambiente de debug
npm init -y

Compile usando
node index.js

--Script do banco de dados--

inicie o Terminal usando
psql -U postgres

Ou ultilize a interfaçe do Postgre
